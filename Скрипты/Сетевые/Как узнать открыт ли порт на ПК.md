# Проверка TCP портов через PowerShell

``` PowerShell
Test-NetConnection IP_АДРЕС -Port НОМЕР_ПОРТА
```

```
PS C:\Users\serge> Test-NetConnection 172.27.155.205 -port 3306
ПРЕДУПРЕЖДЕНИЕ: TCP connect to (172.27.155.205 : 3306) failed


ComputerName           : 172.27.155.205
RemoteAddress          : 172.27.155.205
RemotePort             : 3306
InterfaceAlias         : vEthernet (WSL)
SourceAddress          : 172.27.144.1
PingSucceeded          : True
PingReplyDetails (RTT) : 2 ms
TcpTestSucceeded       : False
```
порт закрыт.

```
PS C:\Users\serge> Test-NetConnection localhost -port 3306
ПРЕДУПРЕЖДЕНИЕ: TCP connect to (::1 : 3306) failed


ComputerName     : localhost
RemoteAddress    : 127.0.0.1
RemotePort       : 3306
InterfaceAlias   : Loopback Pseudo-Interface 1
SourceAddress    : 127.0.0.1
TcpTestSucceeded : True
```
порт открыт



# TelNet

Установка в командной строке с правами администратора
```
dism /online /Enable-Feature /FeatureName:TelnetClient
```

`telnet ip-adress port`

Порт закрыт
```
C:\Users\serge>telnet 172.27.155.205 3306
Подключение к 172.27.155.205...Не удалось открыть подключение к этому узлу, на порт 3306: Сбой подключения
```

Порт открыт
на команду
`C:\Users\serge>telnet localhost 3306`
