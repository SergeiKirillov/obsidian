Как известно, при загрузке Windows 10 не работает клавиша **F8**, отключена возможность выбора дополнительного меню загрузки (безопасный режим, отключение автоматической перезагрузки и др.), как это было сделано в предыдущих версия системах.

Вместо этого Windos 1Х запускает возможность выбора дополнительных параметров загрузки, только после трех аварийных отключений системы или ручным(однократным) запуском в **msconfig**.
![](images/Pasted%20image%2020260211204628.png)

Для включения альтернативного меню при загрузке Windows 10, запустите командную строку с правами администратора:
`ПУСК — Служебные Windows — правой клавишей мыши по Командная строка — Дополнительно — Запуск от имени администратора`

Введите команду:
bcdedit /set {default} bootmenupolicy legacy
