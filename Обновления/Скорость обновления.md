Увеличение пропускной способности для загрузки обновлений Windows (Windows Update) осуществляется ==через настройку «Оптимизации доставки»==, которая позволяет системе использовать больше сетевых ресурсов и загружать файлы с других компьютеров. 
Вот основные способы ускорить этот процесс:

# 1. Настройка параметров Оптимизации доставки (Windows 10/11)
Этот метод позволяет снять ограничения на использование пропускной способности.
1. Откройте **Параметры** > **Центр обновления Windows** > **Дополнительные параметры**.
2. Выберите **Оптимизация доставки**.
3. Убедитесь, что переключатель **Разрешить загрузки с других ПК** включен (чтобы ускорить загрузку за счет локальной сети или интернета).
4. Перейдите в **Дополнительные параметры** (внутри раздела Оптимизация доставки).
5. В разделе «Абсолютная пропускная способность» или «Процент измеренной пропускной способности» установите ползунки на **максимум (100%)** для фоновой и передней загрузки. 
В этом видео показано, как настроить оптимизацию доставки в Windows 10:
https://www.youtube.com/watch?v=ACCuU1lGvqU
# 2. Отключение «Оптимизации доставки» (если интернет быстрый)
Иногда скачивание напрямую от Microsoft работает быстрее, чем через одноранговую сеть (P2P).
1. Перейдите: **Параметры** > **Центр обновления Windows** > **Дополнительные параметры** > **Оптимизация доставки**.
2. Переключите «Разрешить загрузки с других ПК» в положение **Откл**. 
Посмотрите это видео, чтобы узнать, как отключить оптимизацию доставки:
https://www.youtube.com/watch?v=XpoZYXjUY-8&t=12

# 3. Снятие лимита через Редактор реестра (для опытных)

Вы можете отключить троттлинг (ограничение) сети, создав специальный параметр.

1. Нажмите `Win + R`, введите `regedit` и нажмите Enter.
2. Перейдите по пути: `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\DeliveryOptimization`
3. В правой части создайте новый **DWORD (32-bit) Value** с именем `DownloadMode`.
4. Установите значение `0` (или `1` для включения пиринга в локальной сети).
5. Перезагрузите компьютер.
