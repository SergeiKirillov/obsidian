# Коды событий загрузки Windows

| Event ID | Источник                | Описание                           |
| -------- | ----------------------- | ---------------------------------- |
| 12       | Kernel-General          | Система успешно запущена           |
| 13       | Kernel-General          | Система корректно завершила работу |
| 27       | Kernel-Boot             | Ошибка при загрузке                |
| 29       | Kernel-Boot             | Ошибка инициализации загрузки      |
| 30       | Kernel-Boot             | Ошибка загрузчика                  |
| 32       | Kernel-Boot             | Проблема загрузки драйверов        |
| 100      | Diagnostics-Performance | Медленная загрузка системы         |
| 101      | Diagnostics-Performance | Медленный запуск приложения        |
| 102      | Diagnostics-Performance | Медленный запуск драйвера          |
| 109      | Kernel-Power            | Ошибка питания во время загрузки   |

💡 Полезно при:
- долгой загрузке Windows
- циклических перезагрузках
- ошибках загрузчика

# Коды критических ошибок и зависаний

|Event ID|Источник|Описание|
|---|---|---|
|41|Kernel-Power|Аварийное выключение питания|
|55|NTFS|Ошибка файловой системы|
|129|storahci|Таймаут контроллера диска|
|153|disk|Повторная попытка операции чтения|
|157|disk|Диск неожиданно отключился|
|6008|EventLog|Неожиданное выключение|
|7011|Service Control Manager|Таймаут службы|
|7023|Service Control Manager|Служба завершилась с ошибкой|
|7026|Service Control Manager|Драйвер не загрузился|
|1001|BugCheck|Синий экран (BSOD)|

💡 Полезно при:
- **синих экранах**
- зависаниях ПК
- проблемах с **SSD / HDD**
- самопроизвольных перезагрузках

---

✅ **Полезный совет:**  
При диагностике компьютера чаще всего проверяют всего **5 кодов**, которые дают 80% информации:
- **41** — проблема питания
- **6008** — аварийное выключение
- **1000** — сбой программы
- **129 / 153** — проблемы диска
- **219** — ошибка драйвера
